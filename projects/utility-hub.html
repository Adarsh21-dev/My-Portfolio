Utility Hub

Project Description

A multipurpose Android app designed to consolidate various common utilities into a single, convenient application. It includes tools like unit converters, text manipulation utilities, note-taking features, and reminders.

Pros and Cons

Pros:

  • Combines multiple tools, reducing the need for separate apps.
  • User-friendly interface designed with Jetpack Compose.
  • Offline functionality for most features.

Cons:

  • Some niche utility features might be missing compared to dedicated apps.
  • Initial version might have limited customization options.

Tools and Frameworks Used

  • Kotlin
  • Jetpack Compose (UI Toolkit)
  • Android SDK
  • RoomDB (Local Data Persistence)
  • MVVM Architecture Pattern
  • Material Design Components

Features and Functionalities

  • Unit Converter (Length, Weight, Temperature, etc.)
  • Text Utilities (Case Converter, Word Count)
  • Simple Note Pad
  • Reminder System
  • Clean and intuitive UI

Future Enhancements

  • Adding more converter types (Currency, Data Size).
  • Implementing cloud backup for notes and reminders.
  • Adding theme customization options.
  • Calculator functionality.
